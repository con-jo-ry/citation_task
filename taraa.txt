
nanu nirākāram api vijñānam upalabdhilakṣaṇaprāptaṃ svapne 'pi nopalabhyate. tat kathaṃ tad asti paramārthata ity ucyate? ucyate. sukhākāraṃ vijñānam antaḥparisphuradrūpaṃ nirākāraṃ saṃvedyata eva. nīlādyākārāḥ punar alīkāḥ pratibhāsante. anyathā teṣāṃ satyatve sarva evākārāḥ satyāḥ syuḥ. tathā hi grāhyagrāhakabhāvādikam api satyaṃ syāt. tataś ca sarveṣām eva satyapratibhāsatvena muktiprasaṅgaḥ, keṣāñcid api mithyāpratibhāsasya bhrāntirūpasyāpratibhāsanāt. tathā coktam—

	<quote>draṣṭavyaṃ bhūtato bhūtaṃ bhūtadarśī vimucyate |</quote>

tasmād akāmakenāpi nīlādyākārāṇām alīkatvam evaiṣṭavyam. sukhādikaṃ nirākāraṃ satyam upalabhyate. tat kathaṃ nopalabhyata iti.

---

nanv anena nyāyena sakalasākāranirākāravijñānasyālīkatvaprasādhanān na kiñcid api pāramārthikaṃ vastutattvam asti. tat kathaṃ lakṣyasya svarūpaṃ pramāṇata upalakṣayitavyam? naiṣa doṣaḥ, madhyamakamate pramāṇato 'līkatāsiddhāv api māyopamapratibhāsamātrasyaikānekasvabhāvarahitasya dharmirūpasyāpratiṣedhāt. tatraiva cālīke pratibhāsamātre lakṣyalakṣaṇasaṃsāranirvāṇamaṇḍalacakrādibhāvanāsakalajagadarthakriyādīnām avyāhatā vyavasthā sidhyatīti. tathā coktam—

	<quote>buddhatvaṃ vajrasattvatvaṃ saṃvṛtyaiva prasādhayet |</quote>

iti.

nanu sarvam eva vastujātam alīkarūpatayā niḥsāram, tadā kimarthaṃ maṇḍalacakrādibhāvanāprayāsaḥ kriyate? asad etat,

	<quote>mithyādhyāropahānārthaṃ yatno 'saty api moktari |</quote>

iti vacanāt. yady api vicāryamāṇaṃ pāramārthikaṃ vasturūpaṃ nāsti, tathāpy ahaṃ sukhī bhaveyaṃ duḥkhī vā mā bhūvam iti tṛṣṇā sakalaprāṇabhṛtām asti. yathā tulye 'pi mithyātve śubhāśubhasvapnayoḥ śubhasvapnadarśanāt saumanasyam, aśubhasvapnadarśanāc ca daurmanasyam, tadapanayanāya ca saddharmapāṭhamantrajāpādau pravṛttir bhavati, tathā mithyātvāviśeṣe 'pi duḥkhādiprākṛtavikalpahānāya samyaksaṃbodhilakṣaṇaprāptaye ca prekṣāvatām arthināṃ pravṛttir bhaviṣyatīti.

---

prapañcarūpatvād iti cet. prapañcāprapañcayor bhāvanāvasthāyāṃ ko viśeṣaḥ? nanv aprapañcaṃ śīghram eva sthirībhavatīty ayaṃ viśeṣaḥ. yatraivālambane cittaṃ punaḥ punaḥ preryate nirantaraṃ dīrghakālaṃ ca tatraiva sthirībhavatīty āgamo yuktiś cātrāsti. tathā coktam—

	<quote>tasmād bhūtam abhūtaṃ vā yad yad evābhibhāvyate |
	bhāvanābalaniṣpattau tat sphuṭākalpadhīphalam ||</quote>

punaś coktam—

	<quote>aho kusīdatvam aho vimūḍhatā
	aho janasyāsya sadarthavakratā |
	svacittamātrapratibaddhabuddhatā
	adūravartiny api yan na sevyate ||</quote>

iti. tasmān nāyaṃ viśeṣaḥ.

bhrāntirūpatvenāparamārthatvam api sarvatraiva bhāvanāviṣaye vastuni sambhavatīti na kiñcid api bhāvanīyaṃ syāt. tataś ca sarvatraiva mokṣamārge bhāvanāyā vaiyarthyaṃ syāt. māyopamākārānupraveśena bhrāntirūpam apy aprapañcaṃ bhāvyamānam aduṣṭaṃ bhavatīti cet. nanv ayaṃ māyākārānupraveśaḥ prapañce 'pi samāna iti tatrāpi ko doṣasyāvakāśaḥ? tasmāt prapañcam aprapañcaṃ vā yad eva rocate pramāṇasaṃgatam itarad vā, tad evālasyaṃ vihāya mahāmudrārthibhir bhāvayitavyam ity alam atiprasaṅgeneti.

---

na ca nivṛttyā nīrasarūpayā prayojanam asti prekṣāvatām. tathā coktam—

	<quote>mucyamāneṣu sattveṣu ye te prāmodyasāgarāḥ |
	tair eva nanu paryāptaṃ mokṣeṇārasikena kim ||</quote>

iti.

sattvārtho 'pi nivṛttau nāsti. na hi gagane gaganakamale vā kācid arthakriyā sambhavati. ciraniruddhād apy atītād avasturūpāc cakrāt sattvārtho bhaviṣyatīty apy asāram, ciramṛtasyāpi kukkuṭasya kaṇṭhadhvaniprasaṅgāt.
